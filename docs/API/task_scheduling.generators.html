__call__(n_gen, solve=False, verbose=0, save_path=None, rng=None)#

Call problem generator.

Parameters

  • n_gen (int) – Number of scheduling problems to generate.

  • solve (bool, optional) – Enables generation of Branch & Bound optimal solutions.

  • verbose (int, optional) – Progress print-out level. ‘0’ is silent, ‘1’ prints iteration number, ‘2’ prints solver progress.

  • save_path (os.PathLike or str, optional) – File path for saving data.

  • rng (int or RandomState or Generator, optional) – NumPy random number generator or seed. Instance RNG if None.

Yields

  • SchedulingProblem – Scheduling problem namedtuple.

  • SchedulingSolution, optional – Scheduling solution namedtuple.
